# UrbanCove Project

UrbanCove is a Node.js and Express-based web application built using MongoDB and EJS.  
The project follows the MVC (Model–View–Controller) architecture and supports image uploads, authentication, and CRUD operations.

## Technologies Used
- Node.js
- Express.js
- MongoDB (Mongoose)
- EJS
- HTML, CSS, JavaScript
- Cloudinary
- Multer

## Project Structure
controllers/ – Application logic  
models/ – Database schemas  
routes/ – Route handling  
views/ – EJS templates  
public/ – Static files  
utils/ – Helper functions  
uploads/ – File uploads  
app.js – Main server file  

## Setup
1. Install dependencies

2. Create a `.env` file and add required environment variables.

3. Start the server

## Note
The `.env` file is ignored and not pushed to GitHub for security reasons.
